Pentaho NL Usergroup!

Last weekend the European Pentaho User meetup #pcm15 was held in London. Two days of hacking, technical talks and beer was the perfect recipe to make a great meeting. We had great talks, and I caught up with some Dutch colleagues and we concluded there should be some local NL followup to this community initiative! So here it is, a Pentaho NL Usergroup, and it would not be a meetup if there wasn’t any tweettweet… Soon we’ll set out a first date, but please register on meetup! »

Rogier Wessel on #pentaho, #usergroup, 10 Nov 2015
